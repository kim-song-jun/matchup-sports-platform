## Design Memo

이 파일은 MatchUp 디자인의 compatibility memo다.
상세 규칙과 canonical source of truth는 반드시 `DESIGN.md`를 따른다.
문서 읽는 순서와 active/historical 구분은 `docs/DESIGN_DOCUMENT_MAP.md`를 따른다.

### Users
- **타겟**: 20~40대 생활체육 동호인
- **사용 맥락**: 이동 중 빠르게 탐색하고 결정을 내려야 하는 모바일 중심 사용
- **핵심 Job**: 내 수준과 일정에 맞는 상대/팀/활동을 빠르게 찾고 성사시키는 것

### Brand Personality
- **3단어**: 활발 · 스마트 · 친근
- **톤 밸런스**: 친근함 70% + 전문성 30%
- **감정 목표**: 가볍지 않은 신뢰감 위에, 스포츠 서비스다운 에너지와 속도감

### Aesthetic Summary
- **기본 방향**: Toss-like clean layout + MatchUp의 스포츠 에너지
- **핵심 원칙**: layout first, solid first, trust through restraint
- **glass**: chrome only
- **shadow**: restrained only
- **border**: subtle only

### Anti-Patterns
- 과한 shadow, glow, hover lift
- 과한 border, 한쪽 컬러 border, outline-heavy container 중첩
- glass를 본문 카드/폼/리스트 재질로 사용하는 것
- hero-like top zone을 utility page 기본값으로 쓰는 것
- 구조보다 스타일이 먼저 읽히는 card showcase 화면

### Accessibility Baseline
- WCAG 2.1 AA
- 컬러만으로 정보 전달 금지
- 44x44px 터치 타겟
- `prefers-reduced-motion` 대응
