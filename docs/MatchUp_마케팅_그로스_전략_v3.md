## 1. 전략 요약

### Executive Summary

MatchUp은 **국내 생활체육 참여 인구 2,000만 명** 시장을 타겟으로, AI 기반 매칭과 3단계 허위 레벨링 방지라는 차별화된 기술로 기존 수동 매칭 시장의 핵심 페인 포인트를 해결한다.

**핵심 전략 키워드:** 커뮤니티 퍼스트, 데이터 기반 신뢰, 게이미피케이션, 지역 밀착

### 전략 한눈에 보기

| 단계 | 기간 | 목표 | 핵심 전략 |
|------|------|------|-----------|
| **Phase 1** | 0~3개월 | 0 → 1,000명 | 커뮤니티 침투 + 오프라인 |
| **Phase 2** | 3~6개월 | 1,000 → 5,000명 | 바이럴 + 인플루언서 |
| **Phase 3** | 6~12개월 | 5,000 → 30,000명 | 유료 광고 + 제휴 |
| **Phase 4** | 1~2년 | 30,000 → 150,000명 | 전국 확장 + B2B |

### 시장 규모

| 지표 | 규모 | 출처 |
|------|------|------|
| **TAM** (전체 시장) | 생활체육 참여 인구 2,040만 명 | 문화체육관광부 2025 |
| **SAM** (접근 가능 시장) | 풋살/축구/농구/배드민턴 동호인 450만 명 | 대한체육회 |
| **SOM** (초기 목표 시장) | 수도권 풋살/축구 동호인 30만 명 | 자체 추정 |

### MatchUp 핵심 경쟁 우위

1. **AI 매칭**: ELO + 5요소 가중치 알고리즘 — 경쟁사 대비 유일
2. **3단계 레벨링 방지**: 자기기입 → 상호확인 → 사후평가 — 업계 최초
3. **GPS 도착 인증**: 노쇼율을 데이터로 관리 — 경쟁사 없음
4. **S~D 11단계 등급**: 한국 풋살 커뮤니티 표준 차용 — 자연스러운 전환
5. **올인원 플랫폼**: 매칭+채팅+결제+장터+강좌 — 경쟁사 분산

### Phase 1: 커뮤니티 침투 (0~3개월, 목표 1,000명)

#### 2-1-1. 온라인 커뮤니티 타겟팅

| 채널 | 규모 | 진입 전략 | 예상 전환 |
|------|------|-----------|-----------|
| **네이버 카페 "풋살광"** | 회원 5만+ | 유용한 콘텐츠 공유 → 플랫폼 언급 | 200명 |
| **네이버 카페 "풋살인"** | 회원 3만+ | 매칭 사례 후기 공유 | 150명 |
| **네이버 카페 "농구장"** | 회원 2만+ | 농구 매칭 베타 테스트 모집 | 100명 |
| **에브리타임 (대학교)** | 교내 동아리 | 대학 풋살 동아리 직접 컨택 | 100명 |
| **레딧 r/korea** | 외국인 커뮤니티 | 영어 지원 계획 사전 홍보 | 50명 |
| **인스타그램 DM** | 풋살 해시태그 | 활발한 동호인 직접 초대 | 100명 |

**네이버 카페 침투 전략:**
1. 매칭 팁, 구장 리뷰, 레벨 가이드 등 유용한 콘텐츠를 먼저 공유
2. 자연스러운 경험담 형태로 MatchUp 언급 (광고성 느낌 배제)
3. "S~D 등급 측정해보기" 같은 바이럴 콘텐츠 제작
4. 카페 운영진과 협업 (배너 또는 공식 파트너 제안)

#### 2-1-2. 오프라인 활동

| 활동 | 장소 | 방법 | 예상 비용 |
|------|------|------|-----------|
| **구장 방문 홍보** | 서울 주요 풋살장 20곳 | 전단지 + QR코드 + 초대 쿠폰 | 50만원 |
| **무료 경기 주최** | 잠실/마포/강남 | MatchUp 첫 매칭 이벤트 | 100만원/회 |
| **동호인 대회 스폰서** | 지역 아마추어 대회 | 현수막 + 참가자 초대 | 200만원/회 |
| **풋살 구장 제휴** | 10개 구장 | 할인 쿠폰 상호 교환 | 0원 (바터) |

**핵심 오프라인 이벤트: "MatchUp First Match"**
- 매주 토요일 서울 3개 구장에서 무료 매칭 경기 주최
- 참가 조건: MatchUp 가입 + 팀 등록
- 경기 후 즉시 6항목 평가 → 실제 신뢰 지표 체험
- 사진/영상 촬영 → SNS 콘텐츠 활용

#### 2-1-3. 풋살 팀장 직접 영업

풋살/축구 팀의 **팀장(주장)**은 매칭 결정권자이자 확산의 핵심이다.

**팀장 확보 전략:**
1. 네이버 카페/카카오톡 단톡방에서 활발한 팀장 100명 리스트업
2. 1:1 메시지: "기존 카카오톡 매칭보다 편한 시스템 무료 체험"
3. 팀장 전용 혜택: 프리미엄 3개월 무료, 심판 포인트 1,000P 지급
4. 팀장 → 팀원 자연 확산 (1팀 = 10~20명)

**목표: 팀장 50명 확보 = 500~1,000명 사용자**

| **노쇼 사연** | "노쇼 당해서 30만원 날린 이야기" | 카카오 브런치 | 공감 유도 |
| **Before/After** | "카톡 매칭 vs MatchUp 매칭 비교" | 인스타 릴스 | 전환 유도 |
| **팀 소개 영상** | "우리 팀 소개 + MatchUp 전적" | TikTok/Shorts | 팀 가입 유도 |

**바이럴 루프 설계:**

```
사용자 A 가입
  → 팀 등록 + S~D 등급 선택
  → "내 등급 공유하기" (카카오/인스타 스토리)
  → 친구 B가 "나도 해보자" 클릭
  → 사용자 B 가입
  → 반복
```

## 4. 리텐션 & 인게이지먼트

### 4-1. ELO + 뱃지 기반 게이미피케이션

MatchUp의 리텐션 전략 핵심은 **경기할수록 의미 있는 데이터가 쌓이는** 구조이다.

**ELO 레이팅 동기 부여:**

```
신규 가입 → ELO 1,000점 (기본)
    ↓
경기 참가 → 승리 시 ELO 상승 (K=32)
    ↓
등급 변동 → "축하합니다! B+에서 A-로 승급했습니다"
    ↓
프로필 표시 → 다른 사용자에게 공개
    ↓
더 강한 상대 매칭 → 성장 동기 부여
    ↓
반복 참가 (리텐션)
```

**뱃지 수집 동기 부여:**

| 뱃지 | 달성 조건 | 진행도 표시 | 보상 |
|------|-----------|------------|------|
| 매너 플레이어 | 매너 4.5+ | 현재 매너점수/4.5 | 프로필 배지 |
| 시간 약속왕 | 지각률 0%, 10경기+ | X/10 경기 | 프로필 배지 + 1,000P |
| 심판 영웅 | 심판 5회+ | X/5 회 | 프로필 배지 + 쿠폰 |
| 정직한 팀 | 정보 일치도 95%+ | 현재 %/95% | 프로필 배지 |
| 베테랑 | 50경기+ | X/50 경기 | 프로필 배지 + 특별 테두리 |
| 연승 행진 | 5연승+ | 현재 연승/5 | 프로필 배지 |
| 페어플레이 | 무분쟁 20경기 | X/20 경기 | 프로필 배지 |

### 4-2. 주간 리포트 자동 발송

프리미엄 사용자에게 매주 월요일 이메일로 주간 리포트를 발송한다.

**리포트 포함 내용:**

| 섹션 | 내용 |
|------|------|
| **이번 주 전적** | 참가 경기 수, 승/무/패, ELO 변동 |
| **등급 변동** | S~D 등급 변동 사항 + 그래프 |
| **뱃지 진행** | 가장 가까운 뱃지 달성 현황 |
| **매너 점수** | 이번 주 받은 평가 평균 |
| **추천 매치** | 다음 주 추천 매치 3개 |
| **팀 순위** | 내 팀의 지역 순위 변동 |
| **커뮤니티 소식** | 새 강좌, 장터 인기 매물, 이벤트 |

### 4-3. 시즌제 리그
