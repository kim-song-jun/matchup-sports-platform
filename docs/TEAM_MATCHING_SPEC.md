# 팀 매칭 시스템 상세 요구사항

> Teameet 플랫폼의 핵심 기능 — 축구/풋살 팀 간 매칭 + 신뢰 관리 + 경기 후 평가

## 1. 서비스 핵심 문제 해결

| 문제 | 해결 방향 |
|------|----------|
| 조건 맞는 상대 찾기 어려움 | 구조화된 모집글 + 필터 검색 |
| 허위 레벨링 | 3단계 레벨 검증 (자기기입 → 상호확인 → 사후평가) |
| 지각/노쇼 | 도착 인증 + GPS/사진 + 신뢰도 반영 |

## 2. 핵심 플로우

```
1. 팀 회원가입 → 팀 프로필 생성
2. 구장 잡은 팀이 경기 모집글 등록
3. 상대 팀이 모집글에 신청
4. 모집팀이 승인/거절
5. 매칭 성사 → 팀 간 채팅으로 세부 조율
6. 경기 당일 도착 인증
7. 경기 종료 → 결과 + 상호 평가
8. 누적 데이터 → 팀 신뢰도/레벨 신뢰도 관리
```

## 3. 팀 프로필

### 기본 정보
- 팀명, 종목(축구/풋살), 활동 지역
- 주 활동 요일/시간대
- 팀 소개, 평균 연령대
- 유니폼 색상 (홈/어웨이)
- 연락 담당자
- 선수 출신 유무
- 용병 참여 빈도/예정 여부
- 자기기입 레벨 (참고용)

### 신뢰 지표 (누적 데이터)
- 상대평가 레벨 (경기 후 평가 누적)
- 정보 일치도 (선수출신/레벨/용병 사전고지 vs 실제)
- 매너 점수
- 지각률 / 노쇼율 / 취소율
- 최근 경기 수

## 4. 경기 모집글

### 필수 입력
- 종목 (축구/풋살)
- 경기 날짜 + 시작/종료 시간
- 총 경기 시간 (기본 2시간, 변경 가능)
- 쿼터 수 (2/4/6/8/10 선택)
- 구장명 + 구장 주소
- 구장 편의정보 (실내/실외, 주차, 샤워실 등)
- 총 구장비 + 상대팀 부담 비용
- 입금 기한 + 취소 규정
- 상대팀 레벨 조건
- 선수 출신 여부 관련 안내
- 용병 여부
- 경기 스타일 (친선전/실전형/매너 위주)
- 심판 유무
- 특이사항 (자유 메모)

## 5. 심판 자동 배정

심판 없는 경우 쿼터별 심판 담당 팀 자동 배정:
```
4쿼터 예시:
  1Q: A팀 심판 | 2Q: B팀 심판 | 3Q: A팀 심판 | 4Q: B팀 심판
```

### 심판 인센티브
- 심판 봐준 횟수 누적 → 쿠폰/할인 제공
- "매너 플레이어" 뱃지 시스템
- 심판 봉사 포인트 → 구장비 할인 등

## 6. 허위 레벨링 방지 (3단계)

### 1단계: 자기기입 레벨
- 팀 가입 시 직접 입력 (참고용)

### 2단계: 매칭 전 상호 확인
신청/승인 과정에서 확인 체크:
- 선수 출신 유무
- 팀 체감 레벨
- 용병 참여 여부
- 최근 경기 성향 (친선/실전)
- 초보 참여 비중
- "상대 팀 정보를 확인했다" 체크
- "팀 수준 차이에 대해 인지했다" 체크

### 3단계: 경기 후 정보 일치도 평가
- 선수 출신 유무가 실제와 맞았는지
- 팀 수준이 사전 설명과 비슷했는지
- 용병 여부가 사전 고지와 일치했는지
- **누적 데이터 기반 서서히 반영** (감정적 평가 방지)

### 데이터 구조
- 자기기입 레벨
- 상대평가 레벨 (누적 평균)
- 정보 일치도 (%)
- 레벨 신뢰도
- 반복 상대 간 과도한 영향 제한

## 7. 도착 인증 / 지각·노쇼 관리

### 도착 인증
- 경기 당일 팀장이 "도착 완료" 버튼 → 시간 자동 저장
- GPS 기반 구장 반경 인증 (초기: 사진 업로드 보조)
- 상대팀 상호 확인: 정상도착 / 지각 / 미도착

### 판정 기준
| 상태 | 기준 |
|------|------|
| 정상 도착 | 경기 시작 전 도착 |
| 지각 | 시작 후 일정 시간 이내 도착 |
| 노쇼 | 일정 시간 이상 미도착 or 연락두절 |
| 합의 취소 | 사전 합의 후 취소 |

### 조치
- 지각: 경고 + 신뢰 점수 소폭 하락
- 반복 지각: 매칭 신뢰도 하락
- 노쇼: 신뢰 점수 큰 폭 하락
- 반복 노쇼: 매칭 제한/정지

### 팀 프로필 노출 지표
- 지각률, 노쇼율, 취소율

## 8. 참여 유형

### 팀 참여
- 팀 단위로 매칭 신청/모집

### 게스트 참여
- 개인이 팀에 게스트로 합류 신청

### 정기 참여
- 반복 매칭 설정 (매주 토요일 같은 시간 등)

### 용병 구하기
- 인원 부족 시 개별 용병 모집
- 용병 포지션/실력 레벨 지정

## 9. 경기 후 평가

### 평가 항목 (상호 평가)
- 팀 수준이 사전 설명과 일치했는지 (1-5)
- 선수 출신/용병 여부가 사전 고지와 일치했는지 (1-5)
- 매너가 좋았는지 (1-5)
- 시간 약속을 잘 지켰는지 (1-5)
- 비용 정산이 깔끔했는지 (1-5)
- 경기 운영 협조가 원활했는지 (1-5)

### 경기 결과
- 쿼터별 스코어 입력
- 최종 승/무/패

## 10. 뱃지 시스템

| 뱃지 | 조건 |
|------|------|
| 매너 플레이어 | 매너 평가 평균 4.5+ |
| 시간 약속왕 | 지각률 0%, 10경기 이상 |
| 심판 봉사왕 | 심판 5회 이상 |
| 정직한 팀 | 정보 일치도 90%+ |
| 신규 팀 | 가입 후 첫 5경기 |

## 11. 매칭 상태값

```
recruiting    → 모집중
applied       → 신청됨
approved      → 승인됨
rejected      → 거절됨
scheduled     → 경기예정
checking_in   → 도착확인중
in_progress   → 경기중
completed     → 경기완료
cancelled     → 취소
late          → 지각
no_show       → 노쇼
disputed      → 분쟁검토중
```

## 12. 기록 보존 (분쟁 대비)

반드시 저장해야 하는 데이터:
- 신청/승인 시간
- 도착 체크 시간 + GPS 좌표
- 상대 확인 값 (정상/지각/미도착)
- 사진 인증 URL
- 평가 내용 전문
- 신고 기록
- 취소 사유
- 채팅 로그

## 13. MVP 우선 기능

### 필수
- 회원가입/로그인
- 팀 생성 및 프로필 관리
- 경기 모집글 작성/조회
- 상대 팀 신청/승인/거절
- 매칭 성사 후 채팅
- 경기 당일 도착 인증
- 경기 완료 체크
- 경기 후 상호 평가
- 팀 신뢰 지표 저장
- 신고/분쟁 접수

### 보류
- 사이트 직접 구장 선점
- 대기 예약 시스템
- 자동 정산
- 고도화 레벨 알고리즘
- 1/N 결제
- 구장 운영자 전용 페이지
