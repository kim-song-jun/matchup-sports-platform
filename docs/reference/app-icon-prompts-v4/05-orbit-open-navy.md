## 추천 생성 세팅

- 출력 포맷: `PNG`
- 사이즈: `1024x1024`
- 배경: `opaque`
- 스타일: `abstract brand mark`, `open orbit`, `single-color icon`

## 메인 프롬프트

```text
Use case: logo-brand
Asset type: abstract premium app icon concept

Brand context:
An AI sports matching platform that helps real people discover, connect, and play fairly.
The brand should feel precise, trustworthy, and mobile-native.

Primary request:
Design a monochrome app icon concept on a deep navy rounded-square background with one white abstract symbol only.
The symbol should combine an open circular orbit and a short diagonal inner stroke.
It should suggest smart discovery, accurate pairing, precision, and motion without becoming a literal compass, radar, or location pin.
The final silhouette must feel like one iconic sign rather than a utility UI icon.

Background:
Deep navy rounded square.
Flat and premium.

Symbol:
Thick open ring.
Short decisive inner slash.
Breathable opening in the circle.
Balanced spacing.
One unified silhouette.

Shape language:
Calm, precise, modern.
Not sporty in a literal way.
Not dashboard-like.
Not wearable-tech-like.
Strong at small sizes.

Brand reference discipline:
Take only the clarity, reduction, and confidence of Apple, Nike, Adidas, Samsung, and Toss.
Remain fully original.

Constraints:
No sports object.
No map pin.
No gauge marks.
No multiple rings.
No gradient.
No glow.
No text.
No outline.

Output intent:
Centered 1024x1024 PNG brand-icon concept for a mobile app.
```

## 네거티브 프롬프트

```text
compass app icon, location pin, radar interface, gauge dashboard, health app ring, activity tracker, smartwatch icon, gaming HUD, utility UI symbol, fintech dashboard icon, copied nike, copied apple, copied adidas, copied samsung, copied toss
```

## 후속 수정 프롬프트 1

```text
Make the symbol less like a utility icon and more like a brand mark.
Keep the open orbit and short diagonal stroke.
Strengthen silhouette and reduce any interface-like feeling.
```

## 후속 수정 프롬프트 2

```text
Increase timelessness and brand confidence.
Keep the same navy background and white abstract mark.
Do not add complexity.
Improve only ring opening, stroke angle, and visual balance.
```
