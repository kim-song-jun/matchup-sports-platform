## 포지션

- 오빗 계열 중 가장 앱 아이콘스럽고 눈에 잘 띄는 버전
- 밝은 블루 배경 덕분에 홈 화면 시인성이 높다
- 잘 나오면 모노그램보다 더 현대적인 인상을 줄 수 있다

## 디자인 의도

- `오픈 링 + 짧은 사선`이라는 원리는 유지하되, 툴 아이콘처럼 보이지 않아야 한다
- 스포츠 앱 특유의 움직임은 있어야 하지만 실제 스포츠 장비처럼 보여선 안 된다
- `정확한 매칭`, `빠른 탐색`, `기술적 신뢰`를 한 번에 암시해야 한다

## 추천 생성 세팅

- 출력 포맷: `PNG`
- 사이즈: `1024x1024`
- 배경: `opaque`
- 스타일: `blue app icon`, `white abstract mark`, `high-recognition`

## 메인 프롬프트

```text
Use case: logo-brand
Asset type: blue-background abstract app icon concept

Brand context:
A sports matchmaking product for real users, combining precision, trust, and fast discovery in one mobile-first experience.

Primary request:
Create a monochrome blue app icon concept with a vivid blue rounded-square background and one white abstract mark only.
The mark should combine a bold open ring and a short diagonal inner stroke.
It should feel like precision meeting motion, or connection meeting direction.
It must not become a generic radar, a compass, a speedometer, or a software dashboard icon.
The result should feel like a highly distilled brand symbol for a premium modern mobile product.

Background:
Bright blue rounded square.
Clean, flat, confident.

Symbol:
Thick open ring.
Short angled inner stroke.
Strong contrast.
Balanced opening.
One iconic silhouette.

Shape language:
Bold, memorable, clean.
Minimal, but not empty.
Technically sharp without feeling cold.
Readable at small size.

Brand reference discipline:
Use only the abstract lessons of reduction, memorability, and silhouette discipline from Apple, Nike, Adidas, Samsung, and Toss.
Stay original.

Constraints:
No gradient.
No shadow.
No outline.
No extra color.
No sports object.
No multiple rings.
No text.

Output intent:
Centered 1024x1024 PNG app icon concept with strong home-screen presence.
```

fitness tracker ring, smartwatch activity icon, map app symbol, compass, speedometer, crypto icon, software dashboard icon, generic startup logo, navigation UI, copied nike, copied apple, copied adidas, copied samsung, copied toss

```

```
