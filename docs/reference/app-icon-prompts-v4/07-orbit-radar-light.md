## 포지션

- 라이트 배경 오빗형의 기준 후보
- 브라우저 탭, favicon, 웹앱 헤더에서 테스트하기 좋다
- 가장 중립적이고 오래 갈 수 있는 오빗 타입

## 디자인 의도

- `레이더 느낌`을 줄이되 완전히 없애지는 않는다
- 핵심은 `탐색`이 아니라 `브랜드화된 탐색`이다
- 심볼이 도구나 차트처럼 보이면 실패

## 추천 생성 세팅

- 출력 포맷: `PNG`
- 사이즈: `1024x1024`
- 배경: `opaque`
- 스타일: `light background`, `dark abstract mark`, `calm premium tech`

## 메인 프롬프트

```text
Use case: logo-brand
Asset type: light-background abstract app icon concept

Brand context:
A modern sports matching platform with AI, trust, and accurate discovery at its core.
The product tone should be calm, refined, and highly credible.

Primary request:
Design a monochrome icon concept on an off-white rounded-square background with one dark navy abstract symbol.
The symbol should be a refined open orbit with a short inner slash.
It should communicate smart discovery and accurate pairing without becoming a literal radar, compass, map pin, or dashboard element.
The icon should feel more like a timeless product logo than a functional UI symbol.

Background:
Very light off-white rounded square.
Quiet and premium.

Symbol:
Dark navy open ring.
Short, elegant inner slash.
Stable outer weight.
Minimal but memorable.

Shape language:
Calm, premium, restrained.
Clear at small size.
No gimmicks.
No extra geometry.

Brand reference discipline:
Borrow only the high-level lessons of clarity, restraint, and confidence from Apple, Nike, Adidas, Samsung, and Toss.
Do not imitate any existing brand.

Constraints:
No arrows.
No map pins.
No gauge marks.
No interface UI.
No extra circles.
No decorative cuts.
No text.
No gradients.
No texture.

Output intent:
Centered 1024x1024 PNG app icon concept.
```

## 네거티브 프롬프트

```text
compass rose, radar app, location app, speedometer, pie chart, dashboard UI, ring fitness app, software utility icon, analytics chart, copied apple, copied nike, copied adidas, copied samsung, copied toss
```

## 후속 수정 프롬프트 1

```text
Reduce the radar feeling and strengthen the brand mark feeling.
Keep the orbit open and the slash short.
Make the silhouette calmer and more timeless.
```

## 후속 수정 프롬프트 2

```text
Make the icon feel less like a tool and more like a premium consumer product brand.
Do not add new features.
Refine only the ring opening, slash angle, and silhouette weight.
```
