# App Icon Prompts V4

이 디렉토리는 `app-icons-v4`의 각 PNG 시안에 대응하는 개별 프롬프트 문서를 담는다.

사용 방법:

1. 원하는 케이스의 `.md` 파일을 연다.
2. `메인 프롬프트`와 `네거티브 프롬프트`를 함께 사용한다.
3. 첫 결과가 아쉬우면 같은 파일의 `후속 수정 프롬프트`를 이어서 사용한다.
4. 한 번에 모든 케이스를 섞지 말고, 케이스별로 따로 생성한다.

대상 케이스:

- `01-m-round-blue`
- `02-m-compact-navy`
- `03-m-bridge-light`
- `04-m-notch-blue`
- `05-orbit-open-navy`
- `06-orbit-match-blue`
