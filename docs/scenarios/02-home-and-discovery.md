## HOME-003 홈 카드 fallback 이미지가 실사형 로컬 자산으로 유지됨

### Steps

- [x] `/home`에서 매치 카드와 장터 카드가 보이는 상태를 연다.
- [x] API 이미지가 비어 있는 항목의 썸네일을 확인한다.
- [x] 페이지를 새로고침한 뒤 같은 카드의 이미지 슬롯을 다시 확인한다.

### Expected

- [x] fallback 카드가 SVG/아이콘 placeholder가 아니라 실사형 로컬 자산을 노출한다.
- [x] 같은 데이터는 새로고침 후에도 deterministic하게 같은 fallback 계열을 유지한다.
- [x] 카드 비율과 크롭이 깨지지 않는다.
