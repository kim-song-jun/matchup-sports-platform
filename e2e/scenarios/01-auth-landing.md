# 01. Auth, Onboarding & Public Pages

## 페이지 목록

| # | 페이지 | 경로 | 주요 기능 |
|---|--------|------|-----------|
| 1 | Root Redirect | `/` | 토큰 유무에 따라 `/home` 또는 `/landing` 으로 리디렉션 |
| 2 | 로그인/회원가입 | `/login` | 이메일 로그인·회원가입, 소셜 로그인(카카오/네이버), 개발모드 빠른 로그인 |
| 3 | 온보딩 | `/onboarding` | 종목 선택(Step 1) → 기능 소개(Step 2) → 홈 이동 |
| 4 | 랜딩 | `/landing` | 히어로, 통계, 기능 소개, 이용 방법, 종목, 후기, CTA |
| 5 | 소개 | `/about` | 미션, 문제 정의, 해결 방식, 팀 소개, CTA |
| 6 | FAQ | `/faq` | 카테고리 필터 탭, 아코디언 Q&A, 이메일 문의 |
| 7 | 이용 가이드 | `/guide` | 6단계 튜토리얼, 팀 매칭 가이드, 용병/장터 가이드, CTA |
| 8 | 요금 안내 | `/pricing` | 플랜 카드(무료/프로/팀), 매치 참가비, 요금 FAQ, CTA |
