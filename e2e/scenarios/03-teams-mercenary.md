# 03. Teams & Mercenary

## 페이지 목록

| # | 페이지 | 경로 | 주요 기능 |
|---|--------|------|-----------|
| 1 | 팀 목록 | `/teams` | 팀 검색, 종목 필터, 내 팀/다른 팀 구분 |
| 2 | 팀 생성 | `/teams/new` | 팀 등록 폼 (종목 다중 선택, SNS, Shorts URL) |
| 3 | 팀 상세 | `/teams/[id]` | 팀 허브 (소개/굿즈/수강권/대회), 가입 신청, 연락하기 |
| 4 | 팀 수정 | `/teams/[id]/edit` | 팀 정보 수정, 팀 삭제 |
| 5 | 팀 멤버 | `/teams/[id]/members` | 멤버 목록, 역할 변경, 강퇴, 탈퇴, 소유권 이전 |
| 6 | 팀 경기 기록 | `/teams/[id]/matches` | 해당 팀의 팀 매치 목록 |
| 7 | 팀 용병 모집 | `/teams/[id]/mercenary` | 팀 용병 모집글 목록, 관리자 신규 등록 |
| 8 | 용병 목록 | `/mercenary` | 전체 용병 모집글, 종목 필터 |
| 9 | 용병 모집 생성 | `/mercenary/new` | 용병 모집글 작성 폼 |
| 10 | 용병 모집 상세 | `/mercenary/[id]` | 상세 정보, 신청/취소, 신청 관리 |
| 11 | 용병 모집 수정 | `/mercenary/[id]/edit` | 모집글 수정, 삭제 |
