# 05. Chat, Payments & Notifications

## 페이지 목록

| # | 페이지 | 경로 | 주요 기능 |
|---|--------|------|-----------|
| 1 | 채팅방 목록 | `/chat` | 채팅방 목록, 읽지 않은 메시지 뱃지, 데스크탑 2열 레이아웃 |
| 2 | 채팅방 | `/chat/[id]` | 실시간 메시지, Socket.IO, 이모지, 빠른 메시지, 신고/차단/나가기 |
| 3 | 결제 내역 | `/payments` | 탭 필터, 날짜 범위 필터, 결제 목록 |
| 4 | 결제 상세 | `/payments/[id]` | 결제 상태, 타임라인, 영수증 복사, 환불 규정 |
| 5 | 환불 요청 | `/payments/[id]/refund` | 환불 사유 선택, 환불 금액 계산, 확인 모달 |
| 6 | 결제하기 | `/payments/checkout` | 결제 수단 선택, 약관 동의, 매치/강좌 결제 |
| 7 | 알림 | `/notifications` | 알림 목록, 읽음 처리, 전체 읽음, 유형별 아이콘 |
