# 07. Venues, Home, Feed & Other

## 페이지 목록

| # | 페이지 | 경로 | 주요 기능 |
|---|--------|------|-----------|
| 1 | 시설 목록 | `/venues` | 검색·종목 필터·도시 필터·VenueCard 목록 |
| 2 | 시설 상세 | `/venues/[id]` | 허브 탭(소개/굿즈/수강권/대회), 이미지 라이트박스, 예약 일정, 공유 |

| 6 | 평가 | `/reviews` | 미완료 리뷰 목록, 별점 제출 |
| 7 | 뱃지 | `/badges` | 내 뱃지 / 전체 뱃지 탭, 획득·미획득 상태 |
| 8 | 대회 목록 | `/tournaments` | 대회 카드, 상태 뱃지 |

| N | 하단 내비게이션 | (전역, 모바일) | 5개 탭, 더보기 메뉴 시트 |
| N | 사이드바 | (전역, 데스크탑) | 섹션 그루핑 네비게이션, 로그인·로그아웃 |

### A-1. Touch Targets

#### A-1-1. [Critical] VenueCard 썸네일 영역 — 터치 최소 크기 미보장

**파일**: `apps/web/src/components/venue/venue-card.tsx` L29  
```tsx
<div className="relative w-28 shrink-0 bg-gray-100 dark:bg-gray-800 overflow-hidden">
```
VenueCard의 클릭 영역은 `<Link>` 전체지만, 내부 썸네일 컨테이너가 `w-28`(112px) 고정 너비이고 **명시적 최소 높이가 없다**. 카드 전체 높이는 텍스트 콘텐츠 양에 따라 결정되는데, 리뷰/가격 정보가 없는 시설(주소·종목만 표시)은 카드 높이가 44px 미만으로 줄어들 수 있다. CLAUDE.md 기준 "인터랙티브 요소 최소 44×44px".

**수정 방향**: 카드 루트에 `min-h-[68px]` 또는 썸네일에 `min-h-[68px]` 추가.

#### A-1-2. [Warning] HubSectionTab — 최소 높이 없음
